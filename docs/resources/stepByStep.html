Introduction to Curriculum and Learning Platforms

Java JDK 8 and 11

Video tutorial: Setting Up Java JDK 8 and 11

  1. Open file explorer and navigate to C: drive
  2. Add folder: "jba". This will be where you save java library artifacts.
  3. inside folder: "jba", add folder named "java"
  4. Navigate to: Oracle
  5. Select "Java 8 tab"
  6. Select operating system (Windows for AFS Laptops)
  7. Download latest Java 8 JDK for x64
  8. Save to the "java" folder as "java_8_VERSION"
  9. Java 11: follow steps 4-7
  10. *follow video tutorial for installation preferences

Eclipse

  1. Navigate to: Download Eclipse IDE form eclipse.org/downloads
  2. Once you download Eclipse run the eclispe installer exe File.
  3. The Eclipse Installer window will pop up.
  4. Select The Eclipse IDE for Enterprise Java and Web Developers to download.
  5. Next it will take you to a window to install the IDE. Eclipse IDE is only compatible with java 11+. Click the install button.
  6. Once it's done installing click the green launch botton.
  7. Next it will ask you to select a workspace for your projects.

SSH key setup (for Github/Eclipse)

Video Tutorial (1:45-5:12) SSH key and Github Setup for Companion Project

  1. Sign into your github account
  2. Navigate to your settings by clicking on your profile picture in the top right and selecting settings
  3. Under "Access" select "SSH and GPG keys
  4. Select "New SSH key" green button
  5. Under "Title" describe the machine. ex. "AFS Laptop"
  6. Navigate to Eclipse
  7. On the Window tab slect "Preferences"
  8. in the search bar on the left type in "SSH" and under General -> Network Connections, select " SSH2"
  9. Select "Key Management" Tab
  10. Click "Generate RSA Key" button
  11. Copy the SSH key from the text field under "You can paste this public key into the remote authorized_key file:
  12. Navigate back to Github page and paste the SSH key
  13. Navigate back to Eclipse and select "Save Private Key..."
  14. You do not have to password protect the Key
  15. Create the default file path for the SSH folder and save key to that folder
  16. Message should displace successful completion

Github - fork/checkout/feature-branching

Video tutorial (0:00-1:45, 5:12-9:26) fork Repository and checkout

  1. Log into or create Github account
  2. Navigate to ATC-Custom-Bootcamp Java Companion Project
  3. Select Fork button in the upper right corner
  4. Click green "Code" button above the files
  5. Select SSH tab and copy the Git link
  6. Navigate to Eclipse
  7. Open File -> import
  8. In the file directory select Git -> Projects from Git (with smart import)
  9. Select "Clone URI"
  10. Your coppied link from Github should appear in the URI field, if not past the URI there
  11. after selecting next, eclipse will start grabbing the code from Github
  12. deselect branches other than main and click next twice
